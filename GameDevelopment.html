Game systems

I've designed, redesigned, and implemented different game systems to make them efficient, safe, flexible, and scalable, including game economy, progress, audio, and save/load systems.

Diiferent games require different save file settings, I've coded a variety of them, including multiple files, directories, and async systems.

Prototyping

I've made prototypes for mechanics and mini games, focusing on fast testing, adaptability, flexibility, and polish.

I've also made editor components to allow testers to modify values in editor to test different settings on behaviors.

Gamepad implementation

I've implemented gamepad input for different projects with different systems, including Unity's Input system (both old and new), and Rewired plugin. This includes rumbling, and specific platform components like gyroscope, color lights, and adaptive triggers

Back end services

Configuration and coding of cloud and backend services for different platforms: Achievements, store front media and texts, testing environments, publishing settings, and platform specific systems such as PS5 Activities or Steam micro transactions.

UI

Focusing on clarity and efficiency, I've made static, animated, 2D, 3D, screen and world space interfaces, including adaptive systems that show different prompts and graphics according to the current input method.

Debugging & QA

A part of my job has always been to verify everything is running smoothly and working as it should; debugging, QA and bug fixing in different platforms and with different methods, from deep code reviews to exploring the far edges of a scene (both in editor and in-game), I've managed to find and solve different technical issues, crashes, or resource leaks.

Plugin implementation

Different plugins integration and customization to enhance projects or accelerate development: Input, dialogues, animation, or extra functionalities previously inexistent in the game engine.

Project planning and documentation

Analyzing what needs to be done in a project and establishing a roadmap is an important part of development. I redacted techinal quotes for different projects, summarizing the process and ETA needed to develop a feature or port a game to consoles.

I've also contributed to documenting code, development and team's processes to ensure quality and faster development times.
